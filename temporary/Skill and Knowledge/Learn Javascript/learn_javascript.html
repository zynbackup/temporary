* To prevent html form getting submitted when press "enter" in keyboard: 
if(event.which == 13 || event.keyCode == 13){
	return false;
}
Example:
test.html: 
<html>
<body>

<form action="https://www.google.co.id/search" method="get"  onkeypress="if (event.which == 13 || event.keyCode == 13) {return false;}">
  Search on google:<br>
  <input type="text" name="q" placeholder="Input text to search">
  <br>
  <input type="submit" value="Search">
</form>

</body>
</html>
